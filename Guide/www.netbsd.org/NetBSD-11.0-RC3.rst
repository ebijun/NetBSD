**NetBSD 11.0 RC3（2026年4月4日）**
-----------------------------------------------

NetBSD Project は、NetBSD オペレーティングシステムの19番目のメジャーリリースとなる NetBSD 11.0 の**第三のリリース候補版**である **NetBSD 11.0 RC3** を発表しました。

既存のインストールは、インストールイメージを起動して「アップグレード」オプションを選択するか、最初にカーネルとモジュールを更新し、再起動してからユーザー空間を更新することでアップグレードできます。また、パッケージリポジトリのURLを調整し、サードパーティパッケージを更新する必要がある場合があります。


NetBSD 10.1 からの変更点（ハイライト）  
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^  

* **新しいRISC-Vポート:** NetBSD 11.0は、VisionFive 2、PINE64 STAR64、QEMUなどのデバイスを含む64ビットRISC-Vプラットフォームをサポートする最初の安定版リリースです。  
* **POSIX.1-2024** および **C23** プログラミングインターフェース標準への準拠を強化。  
* `compat_linux(8)` における **Linux システムコール**のサポートを強化。これには、epoll、POSIX メッセージキュー、statx、clone3、inotify などが含まれます。  
* **Qualcomm Snapdragon X Elite** プラットフォームの初期サポート。  
* `npf(7)` ファイアウォールの改善。**レイヤー2**と**ユーザー/グループフィルタリング**が含まれます。  
* **x86 向けの新しい MICROVM カーネル**（i386 と amd64 の両方をサポート）。NetBSD 11.0 は、PVH ブート、VirtIO MMIO、および複数のカーネル最適化を活用し、非常に高速な仮想マシン起動を実現するために設計された専用の MICROVM カーネルを導入しました。2020年代の x86 CPU では約 10 ミリ秒で起動できます。  
* **新しい virt68k ポート**。パラバーチャル化されたデバイスを使用して QEMU で Motorola 68000 ポートを実行します。

ポート  
^^^^^^^^^^  

* **aarch64** - Qualcomm Oryon CPU の初期サポート。Qualcomm Snapdragon X Elite のバッテリーおよび充電センサー、GPIO、I2C コントローラーのサポートを追加。  
* **evbppc** - Nintendo Wii の DVD ドライブ、`bwi(4)` WLAN、ハードウェア AES エンジンへのサポートを追加し、X.Orgのパフォーマンスを改善しました。  
* **riscv** - QEMU のサポート、Allwinner D1 SoC、StarFive JH7110 SoC（VisionFive 2、STAR64 などで使用）のサポート、および `crash(8)` カーネルデバッガーのサポートを追加しました。  
* **x86** - 非Xen PVHブートおよびパラバーチャルデバイス用の `pv(4)` 擬似バスのサポートを追加しました。充電動作を制御するための `thinkpad(4)` sysctlインターフェースを追加しました。

カーネル  
^^^^^^^^^^^^^^^  

* **最適化** - SPARC、PowerPC、およびMIPSでのバイトスワップ（特にAdiantumディスク暗号化用）のパフォーマンスを改善しました。  
* **システムコール** - `memfd_create(2)` を追加し、`O_CLOFORK` を実装しました。  
* **デバッグ** - `ddb(4)` にシンプルな show コマンド（condvar、selinfo、sleepq）が追加されました。  
* **ネットワーキング** - `npf(7)` はレイヤー2およびユーザー/グループベースのフィルタリングを獲得しました。  
* **仮想化** - `nvmm(4)` はCPUIDでVMware互換のTSCおよびLAPIC周波数検出を実装しました。  
* **ドライバー** - `drum(4)` はMPSAFEになりました。`wscons(4)` はブライトおよびxtermスタイルの256色エスケープコードのサポートを追加しました。

ユーザースペース  
^^^^^^^^^^^^^^^^^  

* **libc** - `stdio FILE` ハンドルによる書き込みのパフォーマンスを改善。`ffsl(3)`、`mempcpy(3)`、および `timespec_getres(3)` などの関数を追加。  
* **POSIX準拠** - POSIX.1-2024で要求されるISO 2017 Cコンパイラ用の新しい `c17(1)` ラッパースクリプト。`sh(1)` のチルダ展開をPOSIX Issue 8の要件に準拠するように調整しました。  
* **ユーティリティ:**  
* `date(1)` はRFC 5322形式で時刻を表示するための `-R` オプションを追加しました。  
* `df(1)` は `-M` （非マウント引数を無視）および `-q` （警告を抑制）オプションを追加しました。  
* `make(1)` には各種パフォーマンス改善が含まれています。  
* `envstat(8)` はJSON出力のサポートを追加しました。

互換性のない変更点  
^^^^^^^^^^^^^^^^^^^^^^^^  
* このリリースに含まれる新しい **OpenSSH** は **DSA キーをサポートしなくなりました**。  
* 互換性ライブラリ（64 ビットプラットフォーム上の 32 ビットバイナリ用）は、新しい `base32/base64`、`debug32/debug64` セットに分割されました。  
* HTML マニュアルページは、「man」セットから新しい「manhtml」セットに分割されました。  
* `hdaudio(4)` - ミキサーコントロールがソートされ、異なる名前が付けられました。関連するスクリプトを更新する必要があります。  
* `libc` - 実行時にこの API の一般的な誤った使用を検出するために、`ctype(3)` 関数にガードページが追加されました。これにより、以前は間違った結果を返していたコードが**セグメンテーション違反を起こす**可能性があります。古い動作に戻すには、環境変数 `LIBC_ALLOWCTYPEABUSE` を設定してください。  
* `cp(1)` - サイズがゼロに見える場合でも、常に通常のファイルをコピーするようになりました。  
* `mount(8)` - `mount -v spec fs` で出力される情報が削減されました。以前の出力を得るには `-vv` を使用してください。

サードパーティ製コンポーネント  
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^  
多くのサードパーティ製コンポーネントが更新されました。主なものを以下に示します。  

* **binutils** - 2.42 に更新  
* **gcc(1)** - 12.5.0 に更新  
* **gdb(1)** - 15.1 に更新  
* **openssl(1)** - 3.5.1 に更新  
* **ssh(1), sshd(8)** - 10.2 に更新  
* **Xorg(1)** - 21.1.20 に更新  
* **dhcpcd(8)** - 10.2.3 に更新  
* **ACPICA** - 20241212 に更新

NetBSD 11.0 RC3 の入手  
^^^^^^^^^^^^^^^^^^^^^^^^^^^  
NetBSD 11.0 RC3 の完全なソースとバイナリは、世界中の多数のサイトからダウンロードできます。メインの [CDN](https://www.netbsd.org/releases/formal-11/NetBSD-11.0.html) からダウンロードするか、お近くの [ミラーサイト](https://www.netbsd.org/releases/formal-11/NetBSD-11.0.html) をご利用ください。

NetBSD はフリーです。すべてのコードは非制限的なライセンスの下にあり、誰にもロイヤリティを支払うことなく使用できます。メーリングリストやウェブサイトを通じて無料でサポートサービスが提供されています。商用サポートはさまざまな提供元から利用できます。

NetBSD に関するさらに詳しい情報は、[www.NetBSD.org](https://www.NetBSD.org) をご覧ください。

NetBSD 11.0 でサポートされているシステムファミリー  
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^  
NetBSD 11.0 リリースでは、以下のシステムに対してサポートされているバイナリディストリビューションが提供されます。  

* NetBSD/aarch64 (Arm 64-bit)  
* NetBSD/amd64 (AMD Opteron, Athlon64 および EM64T 拡張を備えた Intel CPU)  
* NetBSD/i386 (i486 ファミリー以降のプロセッサを搭載した IBM PC および PC 互換機)  
* NetBSD/riscv (RISC-V ベースのデバイス)  
* その他、多数のプラットフォーム...  

**このリリースに含まれるが、完全にサポートまたは機能していないポート:**  

* NetBSD/ia64 (Itanium ファミリーのプロセッサ)

NetBSD Foundation について  
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^  
NetBSD Foundation は1995年に設立され、NetBSD の中核となるプロジェクトサービスを監督し、産業界やオープンソースコミュニティ内でのプロジェクトを促進し、NetBSD コードベースの多くの知的財産権を保有する役割を担っています。プロジェクトの日常業務はボランティアによって運営されています。

NetBSD Foundation は、商業的な支援を受けていない非営利組織として、ユーザーからの寄付に依存しています。当財団は、優れたオペレーティングシステムの継続的な提供を支援するために、NetBSD Foundation への寄付をご検討いただくようお願い申し上げます。
