## NetBSD RaspberryPi カスタムイメージ作成 仕様

この `Makefile` は、NetBSD-current (HEAD) のバイナリを使用して、**aarch64 (64-bit ARM)** アーキテクチャ向け Raspberry Pi (RPI) 用ブータブルイメージを作成するための設定ファイルです。主に、変数定義とファームウェアの参照情報が含まれています。

## Raspberry Pi (aarch64) NetBSD イメージ作成用 Makefile 仕様書 (設定部)

### 1\. ターゲットアーキテクチャ

| 変数名    | 値         | 説明                    |
| :----: | :-------: | :-------------------- |
| `ARCH` | `aarch64` | ターゲットとする CPU アーキテクチャ。 |

### 2\. イメージファイル名

| 変数名    | 値                                            | 説明                                      |
| :----: | :------------------------------------------: | :-------------------------------------- |
| `FILE` | `date +%Y-%m-%d`\-netbsd-raspi-`${ARCH}`.img | 作成されるイメージファイル名。実行日の日付とアーキテクチャ名を含む。      |
| `ORD`  | `a.img`                                      | (コメントアウトされた行からの推測: 一時的または参照用のイメージファイル名) |

### 3\. NetBSD バイナリソース (NetBSD-current)

| 変数名      | 値                                                                              | 説明                                      |
| :------: | :----------------------------------------------------------------------------: | :-------------------------------------- |
| `BINARY` | `nycdn.netbsd.org/pub/NetBSD-daily/HEAD/20251125125914Z/evbarm-aarch64/binary` | NetBSD の日次ビルド (HEAD) バイナリが置かれているリモートパス。 |
| `SETS`   | `${BINARY}/sets`                                                               | 基本システムセット (.tgz ファイル) が格納されているディレクトリ。   |
| `RPI`    | `${BINARY}/gzimg/arm64.img.gz`                                                 | ベースとなる RPI イメージファイル (圧縮形式)。             |

**【参照情報: NetBSD 10 (コメントアウト)】**

| 変数名      | 値                                                            | 説明                           |
| :------: | :----------------------------------------------------------: | :--------------------------- |
| `BINARY` | `nycdn.netbsd.org/pub/NetBSD-daily/netbsd-10/202404140340Z/` | NetBSD 10 のバイナリパス (コメントアウト)。 |

### 4\. パッケージリポジトリパス

| 変数名        | 値                                                                      | 説明                                 |
| :--------: | :--------------------------------------------------------------------: | :--------------------------------- |
| `PKG_PATH` | `http://cdn.netbsd.org/pub/NetBSD/misc/jun/raspberry-pi/${ARCH}/2025/` | パッケージ管理システムが使用するカスタムパッケージリポジトリのパス。 |

**【参照情報: NetBSD 10 (コメントアウト)】**

| 変数名        | 値                                                                    | 説明                                   |
| :--------: | :------------------------------------------------------------------: | :----------------------------------- |
| `PKG_PATH` | `http://cdn.netbsd.org/pub/pkgsrc/packages/NetBSD/aarch64/10.0/All/` | NetBSD 10 公式パッケージリポジトリのパス (コメントアウト)。 |

### 5\. UEFI (Unified Extensible Firmware Interface) 設定

このセクションは、Raspberry Pi 4 向けの UEFI ファームウェアに関する設定です。

| 変数名         | 値                                                                         | 説明                                             |
| :---------: | :-----------------------------------------------------------------------: | :--------------------------------------------- |
| `UEFI_VER`  | `v1.50`                                                                   | 使用する UEFI ファームウェアのバージョン。                       |
| `UEFI_FIRM` | `RPi4_UEFI_Firmware_${UEFI_VER}.zip`                                      | UEFI ファームウェアの ZIP ファイル名。                       |
| `UEFI_URL`  | `https://github.com/pftf/RPi4/releases/download/${UEFI_VER}/${UEFI_FIRM}` | UEFI ファームウェアのダウンロード元 URL。                      |
| `RPI5_UEFI` | `RPI5`                                                                    | Raspberry Pi 5 向け UEFI の設定値 (コメントアウトされた参照先あり)。 |
